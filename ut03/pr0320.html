Ordenar fechas

Vamos a suponer que tenemos un arrays con cadenas que incluyen fechas de la forma DD-MM-YYYY donde DD es el día, MM es el mes y YYYY es el año.
Desarrolla una función sortDates( arr ) que devuelva un array con las mismas fechas ordenadas cronológicamente.
El objetivo de este ejercicio es prácticas con la manipulación de cadenas y métodos de arrays, por lo que no debes utilizar las funciones de JavaScript para trabajar con fechas.
